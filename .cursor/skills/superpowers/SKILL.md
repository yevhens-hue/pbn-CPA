# AI Superpowers: Advanced Reasoning & Coordination

This skill provides the agent with high-level cognitive frameworks for solving the most challenging engineering problems.

## Instructions

When activated for a complex task, apply these "superpowers":

### 1. First Principles Thinking
- **Action**: Break the problem down into its fundamental truths. Strip away assumptions and previous implementations to find the most direct path to the solution.

### 2. Multi-Agent Coordination
- **Action**: Act as a "Lead Architect" that coordinates multiple subtasks. Use specialized skills (e.g., `code-review`, `web-automation`) as tools in a larger strategic plan.
- **Artifact**: Maintain a highly detailed `implementation_plan.md` that acts as the source of truth for all "agents" involved.

### 3. Deep Root-Cause Analysis
- **Action**: When debugging a "super-bug," trace the failure back through every layer of the stack (infrastructure, network, application logic, database).
- **Technique**: Use the "5 Whys" method to get to the core of the issue.

### 4. Speculative Design
- **Action**: Plan for 6-12 months ahead. Design systems that are not just solutions for today, but foundations for future scaling, observability, and modularity.

### 5. Probabilistic Reasoning
- **Action**: When faced with ambiguity, provide the user with multiple paths, each with an associated "Confidence Score" and "Trade-offs" analysis.

## When to use Superpowers:
- "Refactor the entire authentication system."
- "The site is slow and I don't know why."
- "Migrate this from Python to Go without downtime."
- "Build me a system that can handle 10x traffic."
