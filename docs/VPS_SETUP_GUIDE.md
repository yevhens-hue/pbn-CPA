# 🍎 Руководство для Mac-пользователя: Как запустить Империю

Ваш **MacBook** — это командный центр (Control Center).
Ваши **сайты** — живут на удаленном сервере (VPS).

Вам **НЕ НУЖНО** хостить 50 сайтов на своем Макбуке (он перегреется, и сайты будут недоступны, когда вы закроете крышку).

---

## Шаг 1: Аренда "Земли" (VPS Сервера)
1.  Зайдите на **Aeza.net** (принимают карты РФ/Укр) или **Hetzner/DigitalOcean**.
2.  Купите самый дешевый сервер:
    *   **OS:** Ubuntu 22.04 (или 24.04).
    *   **CPU/RAM:** 2 vCPU / 4GB RAM (хватит на 20-30 легких сайтов).
    *   **Цена:** ~$5-10/мес.
3.  Вам пришлют IP-адрес (например, `45.12.34.56`) и пароль `root`.

## Шаг 2: Подключение с Mac (Terminal)
1.  Нажмите `Cmd + Space` и напишите **Terminal**.
2.  Введите команду:
    ```bash
    ssh root@95.217.163.88
    ```
    *(Это IP вашего нового сервера)*.
3.  Вас спросят, доверяете ли вы серверу — напишите `yes` и Enter.
4.  Вас спросят пароль.
    *   Скопируйте: `dwhJfpPdRbnnUHfC9WNH`
    *   Вставьте (он **не отобразится**, это нормально) и Enter.
5.  **ВАЖНО:** Сервер сразу попросит сменить пароль.
    *   Сначала введите **текущий** (`dwhJfpPdRbnnUHfC9WNH`).
    *   Потом придумайте **новый** (и запишите его!).
    *   Повторите новый пароль.

## Шаг 3: Настройка сервера (Одной командой)
Теперь, когда вы внутри:

1.  Создайте файл установки:
    ```bash
    nano setup.sh
    ```
2.  Скопируйте код скрипта `infrastructure/setup_server.sh` (откройте его на Маке и скопируйте текст).
3.  Вставьте в черное окно терминала (`Cmd+V` или `Ctrl+V`).
4.  Нажмите `Ctrl+O`, `Enter` (сохранить), потом `Ctrl+X` (выйти).
5.  Запустите:
    ```bash
    bash setup.sh
    ```

## Шаг 4: Добавление домена
Допустим, вы купили домен `cricket-aviator.com`.

1.  В панели регистратора (где купили домен) измените **A-запись**:
    *   **Host:** `@`
    *   **Value:** `45.12.34.56` (IP вашего сервера)
2.  На сервере (в Терминале) запустите:
    ```bash
    bash add_site.sh cricket-aviator.com
    ```
3.  **Магия:** Скрипт сам создаст сайт, базу данных и установит WordPress.
4.  Вам выдадут логин/пароль от админки.

## Шаг 5: Подключение к Роботу
1.  Зайдите в WP-админку нового сайта.
2.  Users -> Profile -> Application Passwords.
3.  Создайте пароль `PBN_Bot`.
4.  Скопируйте его и вставьте в нашу **Google Таблицу**.

Всё! Теперь ваш Mac командует, а сервер исполняет.
